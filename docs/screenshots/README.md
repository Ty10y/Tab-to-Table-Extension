# Screenshots

The main [README](../../README.md) expects these image files here:

| Filename | Shows |
| --- | --- |
| `popup-list.png` | The popup listing tab groups (color dot, name, tab count). |
| `copy-success.png` | The popup after copying, with the "Copied N tabs." message. |
| `copied-table.png` | The copied HTML table pasted into a document. |
| `import-view.png` | The Import view with pasted links in the textarea. |
| `imported-group.png` | The resulting new tab group after import. |

Drop PNGs with these exact names into this folder and they'll render in the README.
